String RESTFulAPI.obtainCurrentWeatherSituationHTML ( @PathParam("city") String  city) throws JSONException, IOException

With this method we can obtain the current weather situation of the city we want. To call this method, we only have to request the following URI: http://localhost:8080/CityWeatherWebApp/rest/api/currentWeatherHTML/'NameOfTheCityWeWant'

Parameters
cityThe name of the city which we want to obtain its current weather situation.
Returns
This method returns a HTML file which contains the current weather situation of the city which is passed by parameter.
